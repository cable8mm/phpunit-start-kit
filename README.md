This is a template to create new library.

## Install

```sh
composer require cable8mm/phpunit-start-kit
```

## Usage

```sh
composer create-project cable8mm/phpunit-start-kit <package_name>
```

## Formatting

```sh
composer lint
```

## Test

```sh
composer test
```
